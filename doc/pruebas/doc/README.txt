Compilador adac_cpp.jar (V1.0)
------------------------------
Traduce código fuente en adac a C++, que puede ser compilado y ejecutado.


Invocar como:

-------------------------------------------------------------
java -jar adac_cpp.jar fichero_adac
-------------------------------------------------------------

Si se invoca sin parámetros, lee la entrada estándar.

Genera por la salida estándar la traducción a C++ del fichero suministrado en la invocación.

Ejemplo:
-------------------------------------------------------------
java -jar adac_cpp.jar mcd.adac > mcd.cpp
-------------------------------------------------------------
Generará el fichero "mcd.cpp".
